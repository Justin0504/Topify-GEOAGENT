# Technical SEO Audit Checklist

## Website Speed Optimization

### Core Web Vitals
- **Largest Contentful Paint (LCP)**: Should be under 2.5 seconds
- **First Input Delay (FID)**: Should be under 100 milliseconds
- **Cumulative Layout Shift (CLS)**: Should be under 0.1

### Optimization Actions
- Optimize images (compress, use WebP format)
- Minimize CSS and JavaScript
- Enable browser caching
- Use CDN for static assets
- Optimize server response time
- Implement lazy loading

## Mobile Responsiveness

### Mobile-First Design
- Responsive layout for all screen sizes
- Touch-friendly navigation
- Readable text without zooming
- Fast mobile page load times
- Mobile-friendly forms

### Testing
- Test on multiple devices
- Use Google Mobile-Friendly Test
- Check viewport meta tag
- Verify touch target sizes

## Structured Data (Schema.org)

### Types to Implement
- **Article**: For blog posts and articles
- **Organization**: For company information
- **FAQPage**: For FAQ sections
- **BreadcrumbList**: For navigation
- **Product**: For e-commerce
- **Review**: For ratings and reviews

### Implementation
- Use JSON-LD format (recommended)
- Validate with Google Rich Results Test
- Ensure data accuracy
- Keep structured data updated

## Meta Tags Optimization

### Essential Meta Tags
- **Title Tag**: 50-60 characters, include primary keyword
- **Meta Description**: 150-160 characters, compelling summary
- **Meta Keywords**: Less important, but can include
- **Canonical URL**: Prevent duplicate content
- **Robots Meta**: Control indexing

### Open Graph Tags
- og:title
- og:description
- og:image
- og:url
- og:type

### Twitter Cards
- twitter:card
- twitter:title
- twitter:description
- twitter:image

## Internal Linking Structure

### Best Practices
- Logical site hierarchy
- Clear navigation menus
- Breadcrumb navigation
- Related content links
- Anchor text optimization
- No broken links

### Link Structure
- Homepage links to main categories
- Category pages link to subcategories
- Content pages link to related content
- Use descriptive anchor text
- Maintain reasonable link depth (3-4 clicks max)

## XML Sitemap and Robots.txt

### XML Sitemap
- Include all important pages
- Update regularly
- Submit to search engines
- Use proper priority and frequency
- Include last modification dates

### Robots.txt
- Allow important pages
- Block unnecessary pages
- Specify sitemap location
- Test with robots.txt tester
- Keep updated

## Security (HTTPS)

### SSL Certificate
- Valid SSL certificate installed
- HTTPS redirects from HTTP
- Mixed content issues resolved
- Secure cookies configured
- HSTS header implemented

## Accessibility (WCAG)

### Key Requirements
- Alt text for images
- Proper heading structure
- Keyboard navigation support
- Color contrast compliance
- Screen reader compatibility
- Form labels and error messages

## Technical Issues to Check

### Common Problems
- 404 errors
- Broken links
- Redirect chains
- Duplicate content
- Missing pages
- Server errors (5xx)
- Slow page load times
- Mobile usability issues

## Audit Tools

### Recommended Tools
- Google Search Console
- Google PageSpeed Insights
- Google Mobile-Friendly Test
- Screaming Frog SEO Spider
- Ahrefs Site Audit
- SEMrush Site Audit
- GTmetrix
- WebPageTest

## Audit Report Structure

1. **Executive Summary**: High-level findings
2. **Critical Issues**: Must-fix problems
3. **Important Issues**: Should-fix problems
4. **Minor Issues**: Nice-to-fix improvements
5. **Recommendations**: Actionable steps
6. **Priority Ranking**: What to fix first

## Action Plan

1. Identify critical issues
2. Prioritize by impact
3. Create implementation plan
4. Assign tasks and deadlines
5. Monitor progress
6. Re-audit after fixes
