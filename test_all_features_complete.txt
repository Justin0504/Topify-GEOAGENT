═══════════════════════════════════════
❌ 常见错误检查
═══════════════════════════════════════

如果遇到以下问题，请检查：

1. 【文章内容为空错误】
   - 问题：工具返回 "文章内容为空" 错误
   - 原因：没有先生成文章内容，直接调用了工具
   - 解决：先执行步骤 1，生成完整文章内容，再调用工具

2. 【没有提取到图片】
   - 问题：Word 文档和 WordPress 文章中都没有图片
   - 可能原因：
     * 产品官网没有可提取的图片
     * 图片提取逻辑过滤太严格
     * 网络连接问题
   - 解决：检查产品官网是否有图片，或手动提供 image_urls 参数

3. 【提取了不想要的图片】
   - 问题：提取了 logo、截图或仪表板
   - 可能原因：过滤逻辑需要进一步优化
   - 解决：报告具体提取的图片 URL，以便优化过滤规则

4. 【WordPress 发布失败】
   - 问题：WordPress 发布返回错误
   - 可能原因：
     * 未配置 WP_ACCESS_TOKEN
     * 未配置 WP_SITE_ID
     * API 凭证无效
     * 网络连接问题
   - 解决：检查工具设置中的 WordPress 凭证配置

5. 【图片链接失效】
   - 问题：WordPress 文章中的图片无法显示
   - 可能原因：图片 URL 需要登录或已失效
   - 解决：检查图片 URL 是否可公开访问
